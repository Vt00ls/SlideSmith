# Users own personal workspaces without a tenant layer

SlideSmith's enterprise-intranet first release operates within one organizational trust domain, so each authenticated User owns one Personal Workspace rather than participating through a Tenant layer. Tasks, Artifact Versions, quotas, and usage are owned through that Personal Workspace; cross-user access is explicit and limited to revocable Share Links protected by separate Access Codes. This avoids premature tenant complexity while retaining Workspace as the ownership seam for future team collaboration.
